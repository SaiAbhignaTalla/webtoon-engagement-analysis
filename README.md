# Webtoon Popularity and Reader Engagement

Scraped 1,538 titles from webtoons.com and ran an analysis in Tableau 
to figure out what actually drives readership on the platform.

Tools: Python (scraping), Tableau Public

Dataset: 1,538 titles, 10 attributes. Scraped directly from the platform.
Available here: https://github.com/SaiAbhignaTalla/web_scraping_using_python

Dashboard: https://public.tableau.com/app/profile/sai.abhigna.talla/viz/Talla_FinalProject/Dashboard1#2

Full write-up: findings.md
